WARTUNGSVERTRAG — BRANDMELDEANLAGE — REQUIRED DATA
===================================================

Eine Brandmeldeanlage (BMA) ist eine sicherheitstechnische Anlage zur automatischen
Erkennung und Meldung von Bränden. Die Wartung erfolgt nach DIN 14675 und VDE 0833.

Basis: Wartungsvertrag für Brandmeldeanlagen gemäß DIN 14675 / VDE 0833

REQUIRED FIELDS (must be present)
----------------------------------
- Anlagentyp: Typ / Hersteller / Modell / Anzahl der Brandmeldezentralen und Melder
- Auftraggeber: vollständiger Name, Anschrift, Telefon
- Auftragnehmer: Firma, Anschrift, Telefon
- Objekt der Wartung: Adresse des Wartungsobjekts
- Gegenstand_des_Vertrages: Freitext-Beschreibung der Wartungs-/Instandhaltungspflicht
  (z.B. "Regelmäßige Wartung der Brandmeldeanlage nach DIN 14675 und VDE 0833")
- Vereinbarungen / Leistungen: Liste der vertraglichen Einzelleistungen
  - "Funktionsprüfung aller Brandmelder (automatische und manuelle Melder)"
  - "Inspektion und Prüfung der Brandmeldezentrale inkl. Ausgaben und Anzeigen"
  - "Prüfung aller Alarmierungseinrichtungen (akustisch, optisch, Übertragungsanlage)"
  - "Protokollierung und Dokumentation der Wartung nach DIN 14675"
- Wartungsintervall: z.B. "jährlich", "halbjährlich"
- Laufzeit: { Startdatum (ISO), Enddatum (ISO) ODER "unbefristet",
             Kuendigungsfrist: { frist (z.B. "3 Monate"),
                                 zum (z.B. "Jahresende" | "Monatsende" | "Quartalsende") } }
- Kosten: { Betrag (numerisch), Währung, Bezugseinheit
           (z.B. "pro Wartungsintervall" / "pro Jahr") }
- Gerichtsstand: Ort
- Unterschriften: Liste von { Rolle (Auftraggeber|Auftragnehmer),
                              unterschrift_von, Ort, Datum (ISO) }

OPTIONAL FIELDS (may be present)
---------------------------------
- Reaktionszeit_bei_Stoerung: z.B. "4 Stunden werktags"
- Zusatzleistungen: Liste von Strings
  (z.B. "Entstörungsdienst außerhalb Regelarbeitszeit", "Ersatzteillieferung")
- Zahlungsbedingungen: Beschreibung (z.B. "14 Tage netto")
- Haftungsklauseln: Kurzfassung der Haftungsregelung
- Gewaehrleistungszeit: z.B. "12 Monate"
- Preisanpassungsklausel: Beschreibung
- Mehrwertsteuer: "zzgl. MwSt." | "inkl. MwSt."
- Anlagen_Stueckzahl: Anzahl der zu wartenden Einzelgeräte
- Norm_Pruefung: Norm, nach der die Prüfung durchgeführt wird (z.B. "DIN 14675")
- Anzahl_Melder: Gesamtanzahl der installierten Brandmelder
- Anzahl_Zentralen: Anzahl der Brandmeldezentralen
