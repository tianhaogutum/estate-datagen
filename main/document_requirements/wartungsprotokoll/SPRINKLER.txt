WARTUNGSPROTOKOLL — SPRINKLERANLAGE
=====================================

Ein Wartungsprotokoll für Sprinkleranlagen dokumentiert die planmäßige
Inspektion, Druckprüfung und Funktionsprüfung aller Komponenten einer
ortsfesten Löschanlage gemäß den geltenden Normen und Vorschriften.

Norm: DIN EN 12845

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell der Sprinkleranlage
   (Typ: z.B. "Nassanlage", "Trockenanlage", "Vorgesteuerte Anlage",
    Hersteller: z.B. Minimax/Viking/Tyco, S-Nr., Anzahl Sprinklerköpfe)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   Typische Arbeiten:
     * Druckprüfung Rohrnetz (Betriebsdruck und Prüfdruck messen)
     * Funktionstest Sprinklerköpfe (Sichtprüfung auf Beschädigungen/Korrosion)
     * Inspektion Pumpenanlage (Haupt- und Jockeypumpe, Notstromerzeugung)
     * Prüfung Wasserversorgung und Druckbehälter (Füllstand, Drucklage)
     * Alarmventil prüfen (Alarmdruckschalter, Wassermotor-Gong)
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
   (z.B. "Anlage in ordnungsgemäßem Zustand" oder "Alarmventil undicht, abgedichtet")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
   (das wartungsdurchführende Unternehmen, anerkannter Fachbetrieb)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung, i.d.R. halbjährlich)
- Datum_Inbetriebnahme: ISO-Datum der Erstinbetriebnahme der Sprinkleranlage
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
   (z.B. Sprinklerköpfe, Dichtungen, Manometer, Druckschalter)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
   (z.B. Betriebsdruck_bar, Prüfdruck_bar, Pumpenförderdruck_bar, Tankfuellstand_m3)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
   (z.B. Hauptpumpe, Jockeypumpe)
- Anzahl_Sprinkler_geprueft: Gesamtanzahl der geprüften Sprinklerköpfe (int)
