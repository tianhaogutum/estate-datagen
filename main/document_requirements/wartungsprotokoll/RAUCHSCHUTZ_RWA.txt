WARTUNGSPROTOKOLL — RAUCH- UND WÄRMEABZUGSANLAGE (RWA)
========================================================

Ein Wartungsprotokoll für Rauch- und Wärmeabzugsanlagen (RWA) dokumentiert
die planmäßige Inspektion, Funktionsprüfung und Instandhaltung aller
Entrauchungskomponenten gemäß den geltenden Normen und Vorschriften.

Norm: DIN EN 12101, DIN 18232

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell der RWA
   (Typ: z.B. "Natürliche Rauchabzugsanlage NRA", "Maschinelle Rauchabzugsanlage MRA",
    Hersteller: z.B. D+H/Colt/Lamilux, S-Nr., Anzahl Öffnungsorgane)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   Typische Arbeiten:
     * Funktionsprüfung aller Öffnungsorgane (RWA-Klappen, Lichtkuppeln, Fenster)
     * Steuerungstest (Rauchmelderbetätigung, Handauslösung, Tasterprüfung)
     * Prüfung Auslöseeinrichtungen (Rauchmelder, Thermoschalter, Handauslöser)
     * Messung Entrauchungsquerschnitt (aerodynamisch wirksame Öffnungsfläche)
     * Prüfung Energieversorgung (Netzspannung, Notstromversorgung/Akku)
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
   (z.B. "alle Öffnungsorgane funktionsfähig" oder
    "Antrieb Klappe 2/Dach defekt, Ersatzteil bestellt")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
   (das wartungsdurchführende Unternehmen)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung, i.d.R. halbjährlich)
- Datum_Inbetriebnahme: ISO-Datum der Erstinbetriebnahme der RWA
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
   (z.B. Antrieb, Akku, Steuergerät, Rauchmelder, Handauslöser)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
   (z.B. Oeffnungsquerschnitt_m2 je Öffnungsorgan, Ausloesezeit_s,
    Akkuspannung_V, Steuerstrom_mA)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
- Anzahl_Oeffnungsorgane: Gesamtanzahl der geprüften Öffnungsorgane (int)
