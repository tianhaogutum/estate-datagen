WARTUNGSPROTOKOLL — RAUCH- UND FEUERSCHUTZABSCHLUSS
=====================================================

Ein Wartungsprotokoll für Rauch- und Feuerschutzabschlüsse dokumentiert
die planmäßige Inspektion, Funktionsprüfung und Instandhaltung von
Feuerschutztoren und -rolltoren gemäß den geltenden Normen und Vorschriften.
Ziel ist die Sicherstellung der zuverlässigen selbsttätigen Schließfunktion
im Brandfall sowie die Einhaltung der zugelassenen Einbauzustände.

Basis: ttz-Richtlinie, DIN EN 1634

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell / Seriennummer
   (Typ: z.B. "Feuerschutzrolltor T30", "Feuerschutztor T90", "Rauchschutzabschluss RS",
    Hersteller: z.B. Hörmann/Teckentrup/EFAFLEX, S-Nr., Gesamtanzahl Abschlüsse)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   Typische Arbeiten:
     * Prüfung Schließverhalten (selbsttätiges Schließen, Schließgeschwindigkeit)
     * Auslöseeinrichtung testen (thermische Auslösung und manuelle Notauslösung)
     * Dichtungen prüfen (Umfangsdichtung, Bodendichtung auf Verschleiß und Beschädigung)
     * Laufschienen und Führungen reinigen und schmieren
     * Zulassungskennzeichen und Übereinstimmungserklärung prüfen
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
   (z.B. "alle Abschlüsse schließen ordnungsgemäß" oder
    "thermische Auslösung Abschluss 2/EG träge, Schmelzlot erneuert")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
   (das wartungsdurchführende Unternehmen, zugelassener Fachbetrieb)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung, i.d.R. jährlich)
- Datum_Inbetriebnahme: ISO-Datum der Erstmontage der Abschlüsse
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
   (z.B. Schmelzlot, Umfangsdichtung, Bodendichtung, Auslösemagnet)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
   (z.B. Schliessgeschwindigkeit_m_s, Spaltmass_mm)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
- Pruefbericht_pro_Abschluss: Abschlussweise Prüfdokumentation nach DIN EN 1634
   Liste von { abschluss_id, typ, schliessverhalten, ausloeseverhalten, dichtung_ok }
