WARTUNGSPROTOKOLL — KLIMAANLAGE — REQUIRED DATA
================================================

Ein Wartungsprotokoll für eine Klimaanlage dokumentiert die durchgeführten
Wartungs- und Prüfarbeiten an Kälte- und Klimaanlagen gemäß den geltenden
Vorschriften, insbesondere der EU-Verordnung 517/2014 (F-Gase-Verordnung).

Typische Anlagentypen: Split-Klimaanlage, Multisplit-Anlage, VRF/VRV-System,
Deckenkassette, Truhengerät, Kaltwassersatz

REQUIRED FIELDS (must be present)
----------------------------------
- Anlagentyp: Typ / Hersteller / Modell / Seriennummer / Anzahl der Innengeräte
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
  Typische Arbeitspunkte für Klimaanlagen:
    • Filterwechsel / Filterreinigung (Ergebnis: i.O. | n.i.O. | behoben)
    • Reinigung des Wärmetauschers Innengerät (Verdampfer)
    • Reinigung des Wärmetauschers Außengerät (Kondensator / Verflüssiger)
    • Kältemittelkontrolle (Füllmenge, Zustand)
    • Druckmessung Kältekreis (Hoch-/Niederdruck)
    • Dichtheitsprüfung nach EU-Verordnung 517/2014
    • Kondensatablauf prüfen und reinigen
    • Elektrische Anschlüsse prüfen (Klemmensitz, Kabelzustand)
    • Betriebsgeräusche und Vibration kontrollieren
    • Fernbedienung / Steuerung prüfen
  (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
  (z.B. "Kältemittelfüllmenge geringfügig unter Sollwert, nachgefüllt")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
  Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
---------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung)
- Datum_Inbetriebnahme: ISO-Datum der Erstinbetriebnahme
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
  (z.B. Filtermatten, Kondensatpumpe, Kältemittel R32/R410A)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
  (z.B. Hochdruck bar, Niederdruck bar, Sauggastemperatur °C,
   Überhitzung K, Unterkühlung K, Raumtemperatur °C)
- Kaeltetechnische_Pruefung: { Dichtigkeitskontrolle nach EU 517/2014 (bool),
  Kältemittelart (z.B. R32, R410A, R290), Kältemittelfüllmenge_kg,
  Resultat_Lecksuche_g_a, Wiederholungspruefung_erforderlich (bool) }
  (Pflicht wenn Füllmenge ≥ 5 t CO₂-Äquivalent)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
