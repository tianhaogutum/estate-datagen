WARTUNGSPROTOKOLL — LÜFTUNGSANLAGE — REQUIRED DATA
===================================================

Ein Wartungsprotokoll für eine Lüftungsanlage dokumentiert die durchgeführten
Wartungs- und Prüfarbeiten an mechanischen Lüftungs- und
Wohnraumlüftungsanlagen (mit oder ohne Wärmerückgewinnung) gemäß den
Vorgaben der DIN 1946-6, VDI 6022 und den Herstellerempfehlungen.

Typische Anlagentypen: Zentrale Wohnraumlüftungsanlage mit Wärmerückgewinnung
(WRG), Dezentrale Lüftungsgeräte, Lüftungsanlage ohne WRG, RLT-Anlage
(Raumlufttechnische Anlage), Abluftanlage

REQUIRED FIELDS (must be present)
----------------------------------
- Anlagentyp: Typ / Hersteller / Modell / Seriennummer / Anzahl
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
  Typische Arbeitspunkte für Lüftungsanlagen:
    • Filterwechsel (Zuluft- und Abluftfilter, Filterklasse dokumentieren)
    • Volumenstromprüfung (Zuluft, Abluft, Überströmung je Raum)
    • Reinigung Kanäle und Luftverteilung (Schächte, Rohre, Verteiler)
    • Reinigung Ventilatoren (Zu- und Abluftventilator, Laufräder)
    • Wärmerückgewinnungsgrad messen (Temperaturwirkungsgrad)
    • Steuerung und Regelung prüfen (Einstellparameter, Betriebsmodi)
    • Kondensatableitung kontrollieren und reinigen
    • Außenluft- und Fortluftöffnungen prüfen (Schutzgitter, Vogelschutz)
    • Schalldämpfer und flexible Verbindungen prüfen
    • Erdreich-Wärmetauscher / Außenluftansaugung prüfen (falls vorhanden)
  (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
  (z.B. "Filter stark verschmutzt, getauscht; Volumenstrom nach Filteraustausch i.O.")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
  Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
---------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung)
- Datum_Inbetriebnahme: ISO-Datum der Erstinbetriebnahme
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
  (z.B. Zuluftfilter G4/F7, Abluftfilter G4, Dichtungen, Riemen)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
  (z.B. Volumenstrom Zuluft m³/h, Volumenstrom Abluft m³/h,
   Wärmerückgewinnungsgrad %, Filterdifferenzdruck Pa,
   Zulufttemperatur °C, Ablufttemperatur °C, Außentemperatur °C,
   Fortlufttemperatur °C, Lärmpegel dB(A))
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
  (z.B. Zuluftventilator, Abluftventilator)
