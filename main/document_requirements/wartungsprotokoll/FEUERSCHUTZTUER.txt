WARTUNGSPROTOKOLL — FEUERSCHUTZTÜR
=====================================

Ein Wartungsprotokoll für Feuerschutztüren dokumentiert die planmäßige
Inspektion, Funktionsprüfung und Instandhaltung aller Feuerschutztüren
sowie Feuerschutzabschlüsse gemäß den geltenden Normen und Vorschriften.

Norm: DIN EN 1634, DIN 18095

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell der Feuerschutztüren
   (Typ: z.B. "Feuerschutztür T30", "Feuerschutztür T60", "Rauchschutztür RS",
    Hersteller: z.B. Teckentrup/Hörmann/Jansen, S-Nr., Gesamtanzahl Türen)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   Typische Arbeiten:
     * Prüfung Schließmechanismus und Türschließer (selbstschließende Funktion)
     * Dichtungen prüfen und ggf. tauschen (Umfangsdichtung, Bodendichtung)
     * Türblatt und Zarge kontrollieren (Verformungen, Beschädigungen, Spaltmaße)
     * Zulassungskennzeichen und Übereinstimmungserklärung prüfen
     * Feststellanlage testen (sofern vorhanden, elektromagnetische Feststellung)
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
   (z.B. "alle Türen ordnungsgemäß funktionstüchtig" oder
    "Türschließer Tür 3/OG defekt, ausgetauscht")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
   (das wartungsdurchführende Unternehmen, zugelassener Fachbetrieb)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung, i.d.R. jährlich)
- Datum_Inbetriebnahme: ISO-Datum der Erstmontage der Feuerschutztüren
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
   (z.B. Türschließer, Umfangsdichtung, Bodendichtung, Schlosszylinder)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
   (z.B. Spaltmaß_mm, Schließkraft_N)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
- Pruefbericht_pro_Tuer: Türweise Prüfdokumentation nach DIN EN 1634
   Liste von { tuer_id, feuerwiderstandsklasse, schliessverhalten,
               dichtung_ok, kennzeichen_vorhanden }
