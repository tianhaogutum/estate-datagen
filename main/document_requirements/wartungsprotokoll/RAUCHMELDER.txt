WARTUNGSPROTOKOLL — RAUCHWARNMELDER
=====================================

Ein Wartungsprotokoll für Rauchwarnmelder dokumentiert die planmäßige
Inspektion, Funktionsprüfung und Entstaubung aller installierten
Rauchwarnmelder gemäß den geltenden Normen und Vorschriften.

Norm: DIN 14676

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell der Rauchwarnmelder
   (Typ: z.B. "Optischer Rauchwarnmelder", "Ionisationsrauchwarnmelder",
    Hersteller: z.B. Hekatron/Ei Electronics/Kidde, S-Nr., Gesamtanzahl Melder)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   Typische Arbeiten:
     * Funktionstest per Prüftaste an jedem Rauchwarnmelder
     * Entstauben der Rauchwarnmelder (Reinigung der Raucheintrittskammern)
     * Batterietausch (sofern Batteriekapazität unter Schwellenwert oder Warnanzeige)
     * Protokollierung der Prüfung pro Raum nach DIN 14676
     * Sichtkontrolle Gehäuse auf mechanische Beschädigungen
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
   (z.B. "alle Melder funktionstüchtig" oder "1 Melder in Küche defekt, ausgetauscht")
- Wartungsergebnis: "erfolgreich" ODER "maengel_zu_beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, E-Mail
   (das wartungsdurchführende Unternehmen)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung, i.d.R. jährlich)
- Datum_Inbetriebnahme: ISO-Datum der Erstmontage der Rauchwarnmelder
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
   (z.B. Batterien, Rauchwarnmelder-Komplettgerät)
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
- Pruefberichte_pro_Raum: Raumweise Prüfdokumentation nach DIN 14676
   Liste von { raum, anzahl_rauchmelder, batterietyp, entstaubt,
               prueftaste_geprueft, batterietausch_durchgefuehrt }
