WARTUNGSPROTOKOLL — REQUIRED DATA
=================================

Ein Wartungsprotokoll dokumentiert eine durchgeführte Wartung an einer
technischen Anlage (z.B. Wärmepumpe, Rauchwarnmelder, Lüftungsanlage).

Basis: Beispiel-PDFs aus Daten_AI_Engineer_CaseStudy/
 - Wartungsprotokoll_Waermepumpe-1.pdf  (Glen Dimplex Formular, sehr technisch, mit Checkboxen & Messwerten)
 - Wartungsprotokoll-Rauchwarnmelder-2.pdf (privater Gebrauch, nach DIN 14676, tabellarisch pro Raum)

REQUIRED FIELDS (must be present)
---------------------------------
- Anlagentyp: Typ / Hersteller / Modell der Anlage
   (bei Wärmepumpe: Luft/Wasser / Sole/Wasser / Wasser/Wasser, S-Nr., Softwarestand)
   (bei Rauchwarnmelder: Hersteller, Modell, Anzahl der Melder)
- Anlagenstandort: Adresse und Objekttyp (EFH / MFH / Gewerblich / Wohnung)
- Datum der Wartung: ISO-Datum (YYYY-MM-DD)
- Durchgeführte Arbeiten: Liste der Arbeitspunkte jeweils mit Ergebnis
   (Ergebnis-Werte: "i.O." | "n.i.O." | "behoben" | "ja" | "nein")
- Bemerkungen: Freitext zum Zustand oder besonderen Vorkommnissen
  (z.B. "keine Mängel vorhanden" oder "1 Rauchmelder defekt")
- Wartungsergebnis: "erfolgreich durchgeführt" ODER "durchgeführt, Mängel zu beseitigen"
- Ausfuehrende_Fachfirma: Firma, KD-Techniker, Straße, PLZ, Ort, Tel, Fax, E-Mail
   (das wartungsdurchführende Unternehmen)
- Unterschriften: Liste von { Rolle, Unterschrift_von, Datum }
   Pflicht-Rollen: Techniker und Kunde

OPTIONAL FIELDS (may be present)
--------------------------------
- Auftraggeber: Name und Anschrift
- Naechster_Wartungstermin: ISO-Datum (Fälligkeit nächste Wartung)
- Datum_Inbetriebnahme: ISO-Datum
- Verwendete_Ersatzteile: Liste von { Bezeichnung, Menge }
- Messwerte: Liste von { Bezeichnung, Wert, Einheit }
   (z.B. Vorlauf-/Rücklauftemperatur, Kältemittelfüllmenge, Solekonzentration)
- Historie: Liste von Laufzeitwerten { Komponente, Laufzeit_h }
- Pruefberichte_pro_Raum: Liste von { Raum, Anzahl_Rauchmelder, Batterietyp,
  entstaubt, Verschmutzungen_entfernt, Prueftaste_geprueft, Batterietausch_durchgefuehrt }
- Kaeltetechnische_Pruefung: { Dichtigkeitskontrolle_EU_517_2014 (bool),
  Kältemittelart, Kältemittelfüllmenge_kg, Resultat_Lecksuche_g_a,
  Wiederholungspruefung_erforderlich (bool) }
